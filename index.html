Github Repository

Welcome

This is just a very basic website template for you to use if you want to host a website on Github.io. It's free and super easy. This site contains an index.html, style.css and some images. There are some hyperlinks were necessary, otherwise links are represented as "#". While being simple this template is responsive, fast and uses the bootstrap CDN.

Your Own Website and/or Domain

You can upload this template or any functioning flat (non database) website to github.io using a repository using yourname.github.io or you can even use your own domain name if you wish.
